  <action>Human verification of collaborative editing, cursors, presence, and reconnection across two browser tabs.</action>
  <what-built>Full real-time collaboration system: collaborative TipTap editor with Yjs CRDT sync, Figma-style board cursors with glow effects, online user avatars with animations, Redis-backed presence</what-built>
  <how-to-verify>
1. Start dev server: `pnpm dev`
2. Open two browser tabs to the same board (e.g., http://localhost:5173/board/{demoId})
3. **Cursor test:** Move mouse in Tab A — verify a colored cursor with name pill appears on Tab B's board canvas
4. **Cursor idle:** Stop moving in Tab A for 3+ seconds — verify glow breathe animation starts on Tab B
5. **Online users:** Verify both tabs show avatar circles in the board header
6. **Collaborative editing:** Click a card in both tabs to open the detail modal → type in Tab A's description → verify characters appear in real-time in Tab B's editor
7. **Editor cursors:** Verify remote user's text cursor is visible in the TipTap editor with their color and name label
8. **Floating toolbar:** Select text in the editor → verify floating toolbar appears with Bold/Italic/Strikethrough/Code/Link buttons
9. **Reconnect:** Stop the API server briefly → verify "Reconnecting..." banner appears above the editor → restart server → verify reconnect succeeds
10. **Close tab:** Close Tab A → verify Tab B shows the cursor fade out and avatar leave animation
  </how-to-verify>
