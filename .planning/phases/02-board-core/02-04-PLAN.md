   - `useCreateList(boardId)`: POST /api/boards/:boardId/lists. onMutate: optimistically add list to cache with temp ID and max position + 1000. onError: revert + toast("Couldn't create list. Please try again."). onSettled: invalidate board query.
   - `useUpdateList(boardId)`: PATCH /api/boards/:boardId/lists/:id. onMutate: optimistically update name in cache. onError: revert + toast("Changes couldn't be saved. Please try again."). onSettled: invalidate.
   - `useDeleteList(boardId)`: DELETE /api/boards/:boardId/lists/:id. onMutate: optimistically remove list from cache. onError: revert + toast("Couldn't delete list. Please try again."). onSettled: invalidate.
   - `useCreateCard(boardId)`: POST /api/boards/:boardId/cards. onMutate: optimistically add card to target list. onError: revert + toast("Couldn't create card. Please try again."). onSettled: invalidate.
   - `useUpdateCard(boardId)`: PATCH /api/boards/:boardId/cards/:id. onMutate: optimistic. onError: revert + toast. onSettled: invalidate.
   - `useDeleteCard(boardId)`: DELETE /api/boards/:boardId/cards/:id. onMutate: optimistic remove. onError: revert + toast("Couldn't delete card. Please try again."). onSettled: invalidate.
   
   Each mutation follows D-15 TanStack Query pattern:
   ```typescript
   const queryClient = getQueryClient();
   return useMutation({
     mutationFn: (vars) => apiPost(...),
     onMutate: async (vars) => {
       await queryClient.cancelQueries({ queryKey: ['board', boardId] });
       const previous = queryClient.getQueryData<BoardWithLists>(['board', boardId]);
       queryClient.setQueryData<BoardWithLists>(['board', boardId], (old) => {
         if (!old) return old;
         // apply optimistic change
         return { ...old, lists: ... };
       });
       return { previous };
     },
     onError: (_err, _vars, context) => {
       if (context?.previous) {
         queryClient.setQueryData(['board', boardId], context.previous);
       }
       addToast('error', 'Error message per UI-SPEC copywriting');
     },
     onSettled: () => {
       queryClient.invalidateQueries({ queryKey: ['board', boardId] });
     },
   });
   ```
