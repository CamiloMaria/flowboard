   - `bindState(docName, ydoc)`:
     - Parse cardId from docName (format: `card:{cardId}`)
     - Load card from DB: `prisma.card.findUnique({ where: { id: cardId }, select: { descriptionYjs: true, descriptionText: true } })`
     - If `descriptionYjs` exists: `Y.applyUpdate(ydoc, new Uint8Array(card.descriptionYjs))`
     - If `descriptionYjs` is null but `descriptionText` is not null (D-15 migration): create temporary Y.Doc, get `Y.XmlFragment('description')`, insert text content via `yxmlFragment.insert(0, [new Y.XmlText(card.descriptionText)])`, encode state, apply to ydoc.    - `writeState(docName, ydoc)`:
     - Parse cardId from docName
     - Encode: `const state = Y.encodeStateAsUpdate(ydoc)`

     - Write to DB: `prisma.card.update({ where: { id: cardId }, data: { descriptionYjs: Buffer.from(state), descriptionText: plaintext } })`
   - Export `setupDebouncedPersistence(docName, ydoc, prisma)` function:
     - Tracks a `Map<string, NodeJS.Timeout>` for debounce timers (module-level)
     - On `ydoc.on('update', ...)`: clear existing timer, set new 30-second setTimeout that runs writeState logic
