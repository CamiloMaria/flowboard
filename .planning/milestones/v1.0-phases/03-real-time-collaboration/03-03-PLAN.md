   Note: `y-websocket` (client) is the npm package. Import: `import { WebsocketProvider } from 'y-websocket'`.

2. Create `apps/web/src/hooks/useYjsProvider.ts`:
   - Accept `cardId: string` and `user: { name: string; color: string }` params
   - On mount (useEffect):
     - Create `new Y.Doc()`
     - Get access token via the existing auth mechanism (import `getAccessToken` from lib/api.ts or lib/socket.ts — check which file exports it)
     - Construct WebSocket URL: `const wsUrl = (window.location.protocol === 'https:' ? 'wss:' : 'ws:') + '//' + window.location.host + '/yjs'`
     - Create `new WebsocketProvider(wsUrl, 'card:' + cardId, ydoc, { params: { token: accessToken } })`
     - Set awareness local state: `provider.awareness.setLocalStateField('user', { name: user.name, color: user.color })`
     - Track connection status in state: `provider.on('status', ({ status }) => setStatus(status))` — values: 'connecting', 'connected', 'disconnected'
     - Track awareness changes for co-editors: `provider.awareness.on('change', ...)` → extract remote users from awareness states → update co-editors list
   - On unmount: `provider.destroy(); ydoc.destroy()`
   - Return: `{ ydoc, provider, status, coEditors }` where coEditors is `CoEditorInfo[]`
   - Track reconnect attempt count — after 6 failed attempts, set status to 'failed'

3. Create `apps/web/src/components/editor/CollaborativeEditor.tsx`:

       - `Placeholder.configure({ placeholder: 'Add a description...' })` — D-03

     - No visible toolbar. Formatting via keyboard shortcuts only (Cmd+B bold, Cmd+I italic, Cmd+Shift+S strikethrough, Cmd+E code)
     - Markdown shortcuts enabled by default in StarterKit (`**bold**`, `_italic_`, `` `code` ``)
   - Render:
     ```tsx
     <div className="relative">
       {status === 'connecting' || status === 'disconnected' ? <ReconnectBanner status={status} /> : null}
       {editor && <FloatingToolbar editor={editor} />}
       <EditorContent
         editor={editor}
         className="w-full min-h-[120px] bg-bg-card border border-border-subtle focus:within:border-border-focus rounded-[8px] py-3 px-4 font-body text-sm text-text-primary prose-invert outline-none"
     ```
