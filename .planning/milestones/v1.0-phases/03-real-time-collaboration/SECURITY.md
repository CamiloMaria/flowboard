## Additional Security Observations (Informational)

The implementation includes several hardening measures beyond the declared threat model:

1. **Cursor coordinate validation** (`board.gateway.ts:109-116`): Server validates `x` and `y` are finite numbers before broadcast, preventing NaN/Infinity injection.
2. **Cursor clamping** (`board.gateway.ts:123-124`): Coordinates clamped to 0-10000 range.
3. **Board scope enforcement** (`board.gateway.ts:119-121`): Server validates `data.boardId` matches the board the user actually joined, preventing cross-board cursor injection.
4. **SIGTERM flush** (`yjs-persistence.ts:184-201`): `flushAllDirtyDocs()` ensures no data loss on process shutdown.

These are commendable defense-in-depth measures not required by the threat model.
